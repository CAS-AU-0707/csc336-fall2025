Project Overview (CSC 336 Final Project)
----------------------------------------

What the App Is and What It Does
--------------------------------
My website is a database used to track car collection.
It allows you to input the cars you have based on
mak, model, year, transmission, bodystyle and any notes.

Key features:
- Add new items to a list
- View all items currently in the list
- Items persist between page refreshes
- Hosted online via Render

External Libraries Used
------------------------
Client (React + Vite):
- React: UI framework for handling component structure and state
- Vite: build tool and dev server for fast React development

Server (Node + Express):
- Express.js: handles server routing, API endpoints, and request processing
- CORS: used if cross-origin access is needed
- Nodemon: used during development for auto-restart
